# Rishi Codes — Developer Portfolio 🚀

My personal developer portfolio showcasing full stack projects built with Next.js, Supabase, and AI APIs.

## Live Site
👉 https://rishi-portfolioo.vercel.app

## Projects Featured
- 🤖 AI Resume Analyzer — Next.js + Groq AI + LLaMA 3.3
- ✂️ SnapLink — URL shortener with click analytics
- 💰 SpendSmart — Expense tracker with real-time charts
- 💬 Nexus Chat — Real-time chat with Supabase Realtime

## Built With
- React.js
- Tailwind CSS
- Framer-style CSS animations
- Dark / Light mode toggle
- Fully responsive

## Getting Started
```bash
git clone https://github.com/rishicodes-7/rishi-portfolio
cd rishi-portfolio
npm install
npm run dev
```
