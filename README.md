# &lt;RishiCodes/&gt; — Personal Portfolio

A fast, responsive developer portfolio built with React. Features smooth animations, dark/light mode, particle effects, and a projects showcase.

🔗 **Live Site:** [rishicodes.vercel.app](rishi-portfolioo.vercel.app/)

---

## Preview

![Portfolio Preview](public/resume-ai.png)

---

## Features

- **Dark / Light mode** toggle
- **Animated particle background** on hero section
- **Typewriter effect** cycling through roles
- **Scroll-triggered animations** on every section
- **Fully responsive** — mobile, tablet, and desktop
- **Projects section** with live screenshots and links
- **Skill bars** with animated progress
- **One-click contact** via email

---

## Tech Stack

| Layer | Tech |
|---|---|
| Framework | React |
| Styling | Inline styles + CSS variables |
| Fonts | Space Mono, Bebas Neue, DM Sans |
| Animations | CSS keyframes + Intersection Observer |
| Deployment | Vercel |

## Getting Started

```bash
# Clone the repo
git clone https://github.com/rishicodes-7/portfolio

# Install dependencies
npm install

# Run locally
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
